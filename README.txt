Save your time by quickly creating orders for your woocommerce powered shops, and for existing or new users. Apply a flat discount or a coupon code instantly.

== Description ==

You easily order from your dashboard for new customer & existing customers.

<b>Features:</b>
* Generate order from dashboard
* Create new customer account during generate new order
* Apply flat discount
* Detect existing customers

N.B: If you want more options in this plugin, please [contact with us](https://itech-softsolutions.com/ "iTech Softsolutions")
